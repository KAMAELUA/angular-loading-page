Angular Loading Page comes with an automatic structural directive "mkLoadingPage" for Route changes and Http requests. It add the inner elements if route change or http request start. When all is done, it remove the inner elements.

It also comes with loadings animation components that can be placed within "mkLoadingPage" directive.

You can only use components or directives.

Angular Loading Page is available at npm, if you have an existing application run the following command to download it to your project.

npm install angular-loading-page --save

Angular Loading Page is distributed in commonjs format, a module manager of your choice is required.

UI components are configured as modules, once Angular Loading Page is downloaded and configured, modules and apis can be imported from 'angular-loading-page' shorthand in your application code.

import { LoadingPageModule } from 'angular-loading-page';         //Loading directive
import { MaterialBarModule } from 'angular-loading-page';         //Loading animation component
 
<div *mkLoadingPage [checkPendingRoute]="true" [checkPendingHttp]="true">
  <mk-material-bar colorOne="#dd4b39" colorTwo="#00a65a" colorThree="#f39c12"></mk-material-bar>
</div>
